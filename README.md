## 1 - Автотесты API

Тесты покрывают ключевые сценарии работы всех сервисов:

### CDR Generator
- Генерация CDR файлов
- Проверка структуры CDR

### BRT Service
- Обработка данных о звонках
- Управление абонентами

### HRS Service
- Расчет стоимости звонков
- Управление тарифами

### CRM Service
- Создание абонентов
- Операции с балансом
